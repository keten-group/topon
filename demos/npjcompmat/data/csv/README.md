# CSV exports of `dataset.pkl`

Generated by `convert_pkl_to_csv.py`. Each file is one natural sub-table of the
original pickle. Re-run the script to regenerate after the pickle changes.

Systems: **PDMS**, **PMTFPS**. DP values: **10**, **30**, **100**.

| File | Shape | Format | Description |
|---|---|---|---|
| `mechanics.csv` | 327 x 39 | wide | Per-simulation metrics (mechanical + topological). `folder_name` is the case ID. |
| `analysis_rdf_amorphous.csv` | 641 x 4 | long | Radial distribution function g(r) in the amorphous state. Columns: system, dp, r, g_r. |
| `analysis_rdf_crystal.csv` | 669 x 4 | long | Same as above, crystalline state. |
| `analysis_msid.csv` | 286 x 4 | long | Mean squared internal distance vs. intra-chain separation. Columns: system, dp, n, msid. |
| `analysis_ree_samples.csv` | 1260 x 4 | long | Per-chain end-to-end distance samples. Columns: system, dp, sample_idx, ree. |
| `analysis_rg_samples.csv` | 1260 x 4 | long | Per-chain radius of gyration samples. Columns: system, dp, sample_idx, rg. |
| `tg_msd.csv` | 252 x 13 | wide | Temperature-resolved MSD used for Tg extraction. Columns: system, dp, Temp, MSD_1ps ... MSD_10ps. |

The original `dataset.pkl` remains the authoritative source; these CSVs are for
human inspection and loading from non-Python tools.
